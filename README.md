# Graph-Theory-Demo
Demo some Graph Theory algorithm such as BFS DFS Dijkstra etc. Using WINBGI graphics

# How to build & run:
    + Download DEV-C++ 5.11 (TDM-GCC 4.9.2)
    + Open file 'Graph Theory Demo.dev'
    + Execute > Rebuild all
    + Done
